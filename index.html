Three ways to use it

$

Rip

Disc to MKV in one command. AACS + CSS decryption. All codecs, chapters, subtitles, and audio tracks preserved. HDR10, Dolby Vision, TrueHD, DTS-HD MA.

↔

Stream

Seven stream types. Disc, ISO, MKV, M2TS, network, stdio, null. Any source to any destination. Pipe to ffmpeg. Stream over your LAN. Remux between formats.

↻

Automate

Docker container with web UI. Insert a disc, get an MKV on your NAS. TMDB metadata, poster art, organized into Movies/Title (Year)/. Jellyfin webhooks.

Autorip

Insert a disc. Get an MKV.

Docker container with a built-in web dashboard. Detects disc insertion automatically, rips to MKV, looks up the title on TMDB, organizes into your media library, triggers a Jellyfin scan, and ejects. No interaction needed.

Configurable webhooks for Discord, Jellyfin, n8n, or any HTTP endpoint. Supports MKV, M2TS, ISO, and network output formats.

$ docker run -d --privileged \ -v /mnt/media:/output \ -p 8080:8080 \ ghcr.io/freemkv/autorip:latest

Open localhost:8080. Configure KEYDB URL and TMDB key in Settings. Insert disc.

Install

Linux

curl -sL https://github.com/freemkv/freemkv/releases/latest/download/freemkv-x86_64-unknown-linux-musl.tar.gz | tar xz Download

macOS

curl -sL https://github.com/freemkv/freemkv/releases/latest/download/freemkv-aarch64-apple-darwin.tar.gz | tar xz Download

From Source

cargo install freemkv crates.io
DVD works out of the box. Blu-ray and UHD require a KEYDB.cfg for AACS decryption.

# Rip disc to MKV
freemkv disc:// mkv://Movie.mkv

# ISO to MKV
freemkv iso://disc.iso mkv://Movie.mkv

# Remux M2TS to MKV
freemkv m2ts://Movie.m2ts mkv://Movie.mkv

# Stream over network
freemkv disc:// network://10.1.7.11:9000

# Pipe to ffmpeg
freemkv disc:// stdio:// | ffmpeg -i pipe:0 -c:v libx265 output.mp4
